# 🪣 AWS S3 (Simple Storage Service)

## 📌 What is S3?
S3 is a cloud storage service used to store files like images, videos, backups, and websites.

---

## 🧩 Key Concepts

### 1. Bucket
- Container for storing files

### 2. Objects
- Files stored in bucket

### 3. Storage Classes
- Standard
- Intelligent-Tiering
- Glacier (archive)

---

## 🚀 What I Practiced
- Created S3 bucket
- Uploaded files
- Enabled static website hosting
- Configured public access

---

## 🌐 Use Cases
- Website hosting
- Backup storage
- Media storage

---

## 🔐 Best Practices
- Block public access unless required
- Enable versioning
- Use lifecycle policies

- Hosting static websites
