# 🖥️ AWS EC2 (Elastic Compute Cloud)

## 📌 What is EC2?
EC2 is a virtual server in AWS cloud used to run applications.

---

## ⚙️ Key Features
- Scalable compute capacity
- Pay-as-you-go pricing
- Multiple OS options (Linux/Windows)

---

## 🧩 Key Components

### 1. Instance
- Virtual machine in AWS

### 2. AMI (Amazon Machine Image)
- Template for OS + software

### 3. Security Group
- Firewall for EC2 instance

### 4. Key Pair
- Used for SSH login

---

## 🚀 Steps I Practiced
- Launched EC2 instance (Ubuntu)
- Connected via SSH
- Installed Apache server
- Hosted a simple web page

---

## 🔐 Best Practices
- Restrict security group access
- Stop unused instances
- Use free tier wisely

---

## 🧠 Real-world Use
- Hosting websites
- Running backend servers
- Application deployment
