# 🌐 AWS VPC (Virtual Private Cloud)

## 📌 What is VPC?
VPC is a virtual network in AWS where you can launch and manage AWS resources in a secure and isolated environment.

---

## 🧩 Key Components
- **Subnets** → Divide VPC into smaller networks (Public & Private)
- **Route Table** → Controls traffic flow
- **Internet Gateway** → Allows internet access to VPC
- **Security Group** → Acts as firewall for instances
- **NAT Gateway** → Allows private subnet to access internet securely
